## Pull Request를 올릴때 Issue를 Commit할 때 자동 등록
**키워드 목록**  
* close
* closes
* closed
* fix
* fixes
* fixed
* resolve
* resolves
* resolved  
같은 레포지에서 이슈 삭제하는거  
KEYWORD #ISSUE-NUMBER  
`Closes #10`  
  
이렇게 하면 자동으로 등록된다고 함.
